# 🧠 Multimodal Healthcare AI

AI-powered multimodal disease prediction using **Chest X-ray images** and **Clinical Notes**.

---

## ✨ Features

- 🩻 ResNet50 Vision Encoder
- 📝 BERT Text Encoder
- 🔀 Multimodal Feature Fusion
- 🤖 Disease Classification
- 🌐 Streamlit Web Interface
- ⚡ FastAPI Backend
- 🔥 PyTorch Deep Learning

---

## 🏗️ Model Architecture

- Vision Encoder: ResNet50
- Text Encoder: BERT
- Fusion: Feature Concatenation
- Output: Disease Classification

---

## 🛠️ Tech Stack

- Python
- PyTorch
- Transformers (Hugging Face)
- Streamlit
- FastAPI
- OpenCV
- Pandas
- Pillow
